# Student Management System

## Overview
This project was developed as part of the Alfido Tech Java Developer Internship Program (Task 2).

The Student Management System is a simple console-based Java application that allows users to manage student records through a menu-driven interface.

The project demonstrates the use of Java fundamentals including user input handling, loops, conditional statements, and basic data management.

## Features

- Add Student
- View Students
- Search Student
- Menu-Driven Interface
- User Input Using Scanner
- Conditional Logic and Loops

## Technologies Used

- Java
- Scanner Class
- Visual Studio Code (VS Code)

## Project Workflow

1. Display menu options.
2. Accept user choice.
3. Perform selected operation.
4. Continue until the user exits the program.

## Concepts Used

### Scanner
Used to accept user input from the keyboard.

### Loops
Used to repeatedly display the menu until the user exits.

### Conditional Statements
Used to process user choices and perform specific actions.

### ArrayList
Used to store and manage student records dynamically.

## Sample Functionalities

- Add new student records
- Display all student records
- Search for a student by name
- Exit the application

## Learning Outcomes

- Understanding menu-driven programming
- Practical use of Scanner and ArrayList
- Improved logical thinking and problem-solving skills
- Better understanding of loops and conditional statements
