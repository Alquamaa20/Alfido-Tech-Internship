# Java OOP Concepts Demonstration

## Overview
This project was developed as part of the Alfido Tech Java Developer Internship Program (Task 1).

The objective of this project is to demonstrate the implementation of Object-Oriented Programming (OOP) concepts in Java through simple and practical examples.

## OOP Concepts Implemented

### 1. Encapsulation
- Private data members
- Public getter and setter methods
- Controlled access to data

### 2. Inheritance
- Reusability of code
- Parent and child class relationship

### 3. Polymorphism
- Method overriding
- Runtime behavior demonstration

## Technologies Used
- Java
- Visual Studio Code (VS Code)

## Project Structure

- EncapsulationDemo.java
- InheritanceDemo.java
- PolymorphismDemo.java

## Learning Outcomes
- Understanding of OOP principles
- Hands-on practice with Java classes and objects
- Improved knowledge of code reusability and data security
